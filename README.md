# Memory Card Game 🧠🎴

A modern and responsive memory card matching game built with React.  
The goal is to match all card pairs in the fewest moves and shortest time possible.

## 🚀 Features
- 4x4 memory card grid
- Smooth flip card animations
- Move counter and game timer
- Game completion screen
- Game history saved in localStorage
- Responsive design for desktop and mobile
- Clean and modern UI

## 🛠 Tech Stack
- React
- JavaScript (ES6+)
- CSS
